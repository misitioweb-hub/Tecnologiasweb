¿Qué es CSS3?

CSS3 es el lenguaje que usamos para diseñar páginas web. Define colores, tamaños, posiciones, animaciones, entre otros.

Características nuevas de CSS3:

  • Bordes redondeados con border-radius
  • Sombreados con box-shadow
  • Animaciones y transiciones

Ejemplo:

div {
  background: lightblue;
  border-radius: 10px;
  padding: 20px;
  transition: all 0.5s;
}
